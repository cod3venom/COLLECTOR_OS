
<h1> Collector OS </h1>

<p>Collector OS is a lightweight linux kernel based operating system for portable data collectors.<p>

## Features

- [x] Bluetooth scanner
- [X] Barcode scanner
- [X] Wifi scanner
- [X] SqlLite based local database
